# Tessera

A portfolio-grade mid-frequency ML trading system for crypto perpetual futures
(Binance + Bybit, USDT-margined). Holding period 15 minutes to 4 hours, with a
secondary delta-neutral funding-rate carry sleeve.

## Quick Start

```bash
make setup    # install deps + pre-commit hooks
make lint     # ruff + mypy --strict
make test     # pytest with coverage
make docs     # serve MkDocs locally
```
